# Domain Glossary

- `Primary app path`:
  browsing/searching games, game details, recommendation loading.
- `Convention hours`:
  active event window where warm runtime profile is enabled.
- `Degraded mode`:
  app remains available while recommendation artifacts are unavailable.
- `Release traceability`:
  mapping of git SHA/tag to Fly release and validation evidence.
- `Contract ownership`:
  backend API/schema is canonical; frontend follows by explicit integration updates.
- `Embedding artifacts`:
  recommendation model data files (for example sparse matrix `.npz` and index mapping JSON) loaded by recommendation paths.
- `LibraryGame linkage`:
  mapping between convention game records and canonical board game records (typically via `bgg_id`).
